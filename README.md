
# Trackora

Trackora is a full-stack web application built to help students stay organized throughout their placement journey. The platform combines coding preparation analytics and internship application management, allowing users to track solved problems, identify weak topics, monitor consistency, manage application statuses, and gain actionable insights from a single dashboard.


## Problem Statement

Students often use multiple tools to track internship applications, coding practice, and placement preparation. This scattered workflow makes it difficult to stay organized, monitor progress, identify weak areas, and keep track of important deadlines.

Trackora solves this by bringing internship tracking and coding preparation into a single platform, helping students stay focused and manage their placement journey more effectively.


## Core Features

📊 Dashboard

A centralized dashboard that provides an overview of coding preparation and internship applications, along with key insights and progress metrics.

🔐 Authentication

Secure user authentication allows users to create accounts, log in, and access their personalized data through a protected dashboard experience.

💻 Preparation Tracker

Track solved coding problems, monitor daily goals and interview readiness, and identify weak topics through preparation analytics.

🎯 Internship Tracker

Manage internship applications by adding, editing, deleting, and tracking statuses and deadlines in one place.

📈 Progress Analytics

Measure coding progress, track hard problem completion, monitor consistency, and gain actionable insights to improve preparation.


🎨 User Experience

Built with a responsive and intuitive interface that provides real-time updates and a seamless user experience across all modules.
## Tech Stack

Frontend: React, Tailwind CSS, React Router

Backend: Node.js, Express.js

Database: PostgreSQL, Prisma ORM

Tools & Libraries: Lucide React, Git, GitHub

## Future Enhancements

Future versions of Trackora will focus on providing deeper insights and a more personalized placement preparation experience.
